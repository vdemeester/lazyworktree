Terminal-first Git worktree control

Easy worktree management for the terminal.

LazyWorktree is a fast TUI for creating, reviewing, and switching Git worktrees. It combines Vim-style navigation, PR and CI visibility, and deep Git tooling in one focused interface.

Download Releases Read the Docs

Quick start

$ brew tap chmouel/lazyworktree https://github.com/chmouel/lazyworktree
$ brew install lazyworktree --cask
$ cd your-repository
$ lazyworktree

Features

Manage your worktree with Ease.

Worktree management

Create, rename, remove, absorb, and prune merged worktrees directly from the keyboard.

Powerful creation modes

Create from branch, commit, PR/MR, issue, or custom menu item with optional branch automation.

Deep Git tooling

Stage, unstage, commit, diff, sync, push, and cherry-pick without leaving the interface.

CI visibility

Display checks, open job pages, inspect logs in pager, and restart GitHub Actions jobs.

Session aware

Manage tmux or zellij sessions per worktree with configurable session prefixes.

Extensible automation

Run init and terminate hooks via .wt and add custom command bindings.

Taskboard from notes

Turn markdown checkbox notes into a grouped Taskboard and toggle progress without leaving the TUI.

Install

Get running in less than one minute.

brew tap chmouel/lazyworktree https://github.com/chmouel/lazyworktree
brew install lazyworktree --cask
yay -S lazyworktree-bin
go install github.com/chmouel/lazyworktree/cmd/lazyworktree@latest

Optional tools: gh or glab for forge data, and delta for enhanced diffs.
